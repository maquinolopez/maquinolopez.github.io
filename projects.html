üç™ BISCUIT

GitHub Repository

BISCUIT (Bayesian Integration of Sequences Using Chronology Intersection Tie-points) is an R package that enables probabilistic alignment of an input paleoclimate record with a target record that has known chronological control points. It combines:

  • Posterior age distributions at tie-point depths in the target record
  • Prior knowledge of sediment accumulation and memory

This results in a fully Bayesian age-depth model for the input sequence.

üåÆ TACO

GitHub Repository

TACO (Transitional Analysis through Clymo Optimization) introduces a novel Bayesian framework to understand complex carbon dynamics in peatland ecosystems. Extending the classic Clymo model, it supports:

  • Smooth transitions between carbon states
  • Variable carbon influx modeling
  • Inference on deep-layer decomposition processes

TACO provides a nuanced and flexible modeling tool for biogeochemical dynamics.
